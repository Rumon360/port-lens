### 📦 Publish Guide (Electron Forge + GitHub)

1. Open **Command Prompt (CMD)**

2. Set your GitHub token:

```
set GITHUB_TOKEN=your_token_here
```

3. Run publish:

```
npm run publish
```

---

### ✅ Notes

- Make sure your token has **repo (full access)** or **Contents: Read & Write**
- This command only sets the token for the current CMD session
- You must run both commands in the **same window**
